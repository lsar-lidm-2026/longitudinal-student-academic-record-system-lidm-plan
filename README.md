# Longitudinal Student Academic Record

Lomba Inovasi Digital Mahasiswa (LIDM) 2026

Pengembangan Sistem Longitudinal Student Academic Record Berbasis Artificial Intelligence untuk Mendukung Administrasi Akademik dan Penyusunan Buku Induk Siswa Sekolah Dasar.

## Project Overview

Proyek ini bertujuan membantu guru sekolah dasar dalam mengelola riwayat akademik siswa secara longitudinal sehingga proses administrasi dan penyusunan Buku Induk menjadi lebih mudah.
